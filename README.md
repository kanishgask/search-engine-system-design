# 🔎 Search Engine System Design

System Design Project – Information Retrieval System

---

# Problem Statement

Design a scalable search engine similar to Google that can crawl websites, build indexes, and return relevant search results.

---

# Functional Requirements

- Crawl websites
- Index documents
- Search documents
- Rank results

---

# Non Functional Requirements

- High scalability
- Fast search response
- Fault tolerance
- Distributed storage

---

# Core Concepts

• Web Crawling  
• Inverted Index  
• Search Ranking  
• Query Processing  

---

# High Level Architecture

User
 ↓
API Gateway
 ↓
Search Service
 ↓
Index Service
 ↓
Document Store
